> Node module for ajency tasks


## Getting Started

Install the module with: `npm install ajency`

```js
var ajency = require('ajency');
ajency.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g ajency
$ ajency --help
$ ajency --version
```


```sh
# creates a browser.js
$ grunt browserify
```
